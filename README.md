# 🌦 Weather Dashboard Pro

A modern weather application built with **Python**, **CustomTkinter**, and the **OpenWeatherMap API** that provides real-time weather information through an interactive desktop interface.

## 🚀 Features

- Real-time weather updates
- Temperature & Feels Like
- Humidity, Pressure & Wind Speed
- Visibility & Cloud Coverage
- Sunrise & Sunset Time
- City & Country Information
- Modern Dark-Themed UI

## 🛠 Technologies Used

- Python
- CustomTkinter
- Requests
- OpenWeatherMap API

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python weather_app.py
```
